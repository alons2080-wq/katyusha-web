El instalador
del futuro

Katyusha es un instalador de paquetes para Linux, rápido, modular y con escáner de seguridad integrado. Inspirado en APT, Pacman y las herramientas modernas.

bash — katyusha

¿Por qué Katyusha?

Potente, seguro y extensible

🛡️

Escáner de Seguridad

Motor de análisis en Rust que detecta posibles amenazas de malware antes de instalar cualquier paquete.

⚡

Motor en C

Frontend principal escrito en C para máxima velocidad y control sobre el sistema de instalación.

📦

Multi-formato

Soporta .deb, .AppImage, .zip y .tar.gz. Instala desde rutas locales o URLs remotas sin complicaciones.

🌐

Repositorios Remotos

Conecta con tu propio índice privado en GitHub. Configura el token y la URL, y tendrás tu propio repositorio.

⚠️

Alertas Inteligentes

Detecta archivos grandes (1GB–9GB) y solicita confirmación antes de proceder. Sin sorpresas.

🧩

Arquitectura Modular

Separación clara de responsabilidades. Fácil de extender, modificar y adaptar a nuevos formatos.

Comenzar

Compilar e instalar

mkdir -p build && cd build
cmake ..
cmake --build .
cd rust/scanner
cargo build --release
make package
sudo dpkg -i build/katyusha_1.0.1_amd64.deb
katyusha install paquete.deb
katyusha install https://example.com/app.AppImage
katyusha install --temp paquete.deb
katyusha update
katyusha list
katyusha upgrade paquete

Estructura del repositorio

Katyusha/

 ├── build/        ← Salida de compilación CMake
 ├── tests/        ← Pruebas y casos futuros
 ├── .devcontainer/ ← Config contenedor de desarrollo
 └── CMakeLists.txt
